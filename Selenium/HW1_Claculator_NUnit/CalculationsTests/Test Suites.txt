В NUnit 3.0 и выше аттрибут Suite был изъят.

Соответственно для запуска отдельных сетов тестов используется nunit-консоль, где запускается тестируемая библиотека, содержашая тесты,
 и условия, по которым они фильтруются для данного рана.
В случае данного проекта для полного запуска тестов можно использовать условие namespace, для частичных - отбор по категории.
Соответственно,

nunit3-console CalculationsTests.dll --where "namespace == CalculationsTests"

nunit3-console CalculationsTests.dll --where "cat == Rising"
